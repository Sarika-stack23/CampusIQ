----------------------------------------------------------------
  PRODUCT IDENTITY
----------------------------------------------------------------

Name        : CampusIQ
Tagline     : "The only app that knows your entire college life — not just your tasks."
Target      : All Indian college students (Engineering, Medical, Arts, MBA)
Stack       : MongoDB + Express + React + Node.js (MERN)
State Mgmt  : Zustand
Auth        : JWT (JSON Web Tokens)
AI          : OpenAI API (Phase 3 onwards)
Payments    : Razorpay (Phase 4)
Deploy      : Vercel (frontend) + Render or Railway (backend) + MongoDB Atlas (DB)

================================================================
  ALL API ROUTES
================================================================

AUTH
  POST  /api/auth/register       Create new account
  POST  /api/auth/login          Login, receive JWT token
  GET   /api/auth/me             Get currently logged in user

TASKS
  GET   /api/tasks               All tasks (filterable by status, priority)
  POST  /api/tasks               Create new task
  PATCH /api/tasks/:id           Update task (status, deadline, etc)
  DEL   /api/tasks/:id           Delete task

ATTENDANCE
  GET   /api/attendance          All subjects with percentages
  POST  /api/attendance          Add new subject
  PATCH /api/attendance/:id      Mark present or absent

EXAMS
  GET   /api/exams               All exams with countdowns
  POST  /api/exams               Add new exam
  PATCH /api/exams/:id           Update exam

EXPENSES
  POST  /api/expenses            Log new expense
  GET   /api/expenses            All expenses (filterable by month)
  GET   /api/expenses/summary    Monthly breakdown by category

SLEEP
  POST  /api/sleep               Log sleep entry
  GET   /api/sleep?days=30       Last N days of sleep data

RATINGS
  POST  /api/ratings             Submit new rating (mess / prof / hostel)
  GET   /api/ratings/:college/:type  Get all ratings for a college by type

PLANNER
  GET   /api/planner/today       Get or generate today's study plan
  POST  /api/planner/generate    Manually trigger plan regeneration

AI
  GET   /api/ai/insights         Burnout score + pressure index + suggestions

================================================================
  DEPLOYMENT GUIDE
================================================================

FRONTEND — Vercel
  1. Push /client to GitHub
  2. Go to vercel.com, import your repo
  3. Set build command: npm run build
  4. Set output directory: dist
  5. Add environment variable: VITE_API_URL = your Render backend URL
  6. Deploy. Auto-deploys on every git push.

BACKEND — Render
  1. Push /server to GitHub
  2. Go to render.com, create new Web Service
  3. Connect your GitHub repo
  4. Set start command: node src/server.js
  5. Add environment variables: PORT, MONGO_URI, JWT_SECRET, OPENAI_API_KEY
  6. Deploy. Free tier available.

DATABASE — MongoDB Atlas
  1. Go to mongodb.com/atlas
  2. Create free M0 cluster
  3. Create database user with password
  4. Whitelist IP: 0.0.0.0/0 (allow all for now)
  5. Copy connection string into .env as MONGO_URI
